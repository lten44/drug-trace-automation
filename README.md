# 国瑞新特药追朔码自动处理软件

[![Python](https://img.shields.io/badge/Python-3.12+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

> 一个用于自动化处理药品追溯码的桌面应用程序，通过模拟鼠标操作与"码上放心"客户端交互，实现追溯码的自动查询和1级码提取。

## 📖 项目简介

本软件专为药品批发企业设计，解决日常工作中大量追溯码处理的痛点。用户只需将Excel表格中的追溯码导入，软件即可自动：

1. 打开"码上放心"客户端
2. 逐个查询追溯码的关联关系
3. 提取所有1级追溯码
4. 生成标准格式的输出表格

**适用场景**：
- 每天需要处理数百至数千条追溯码
- 需要为医院提供1级追溯码表格
- 手工操作效率低、易出错

## ✨ 核心功能

- 🔍 **自动查询**：模拟鼠标操作，自动在"码上放心"中查询追溯码
- 📝 **智能提取**：自动过滤并提取1级追溯码，排除2级/3级码
- 📊 **批量处理**：支持Excel批量导入，后台线程处理不卡顿
- 🎨 **友好界面**：卡片式UI设计，进度条实时显示
- 🔄 **自动更新**：内置更新机制，打开即是最新版本
- ⚙️ **灵活配置**：支持自定义默认输入/输出路径

## 🚀 快速开始

### 系统要求

- Windows 10/11
- Python 3.12+（开发环境）
- "码上放心"客户端（已安装并登录）

### 安装使用

#### 方式一：使用预编译版本（推荐）

1. 下载 `国瑞新特药追朔码自动处理软件.exe`
2. 放置到 `E:\国瑞新特药追朔码自动处理软件\` 目录
3. 创建桌面快捷方式
4. 双击运行

#### 方式二：从源码运行

```bash
# 克隆仓库
git clone https://github.com/yourusername/drug-trace-automation.git
cd drug-trace-automation

# 安装依赖
pip install -r requirements.txt

# 运行
python gui.py
```

### 首次使用步骤

1. **校准位置**：点击"校准位置"按钮，依次点击：
   - 追溯码输入框
   - 查询按钮
   - 一键复制所有码（首次位置）
   - 一键复制所有码（查询后位置）
   - 产品批号文字位置

2. **选择文件**：点击"浏览"选择要处理的Excel文件

3. **开始处理**：点击"开始处理"按钮，等待完成

## 🛠️ 技术栈

- **GUI框架**：Tkinter（Python内置）
- **自动化**：PyAutoGUI（鼠标/键盘模拟）
- **Excel处理**：OpenPyXL
- **剪贴板**：Pyperclip
- **打包工具**：PyInstaller
