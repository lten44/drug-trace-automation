# 药品批发企业追朔码自动处理软件

[![Python](https://img.shields.io/badge/Python-3.12+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

> 一个用于自动化处理药品追溯码的桌面应用程序，通过模拟鼠标操作与"码上放心"客户端交互，实现追溯码的自动查询和1级码提取，生成标准化输出表格。

**当前版本**：v2.2 | **运行平台**：Windows 10/11 | **打包方式**：PyInstaller 单文件 exe

---

## 📖 项目简介

本软件专为药品批发企业设计，解决日常工作中大量追溯码处理的痛点。用户只需将Excel表格中的追溯码导入，软件即可自动：

1. 打开"码上放心"客户端
2. 逐个查询追溯码的关联关系
3. 提取所有1级追溯码
4. 生成标准格式的输出表格

**适用场景**：
- 每天需要处理数百至数千条追溯码
- 需要为医院提供1级追溯码表格
- 手工操作效率低、易出错

## ✨ 核心功能

- 🔍 **自动查询**：模拟鼠标键盘操作，自动在"码上放心"中查询追溯码
- 📝 **智能提取**：自动过滤并提取1级追溯码，排除2级/3级码
- ⚡ **智能跳过（v2.1）**：码包装数为1的行自动识别为1级码，跳过查询和过滤步骤，处理速度提升50%以上
- 📊 **批量处理**：支持Excel批量导入，后台线程处理不卡顿
- 🎨 **友好界面**：卡片式UI设计，进度条实时显示
- 🔄 **自动更新**：启动时自动检查版本，发现新版本自动覆盖重启
- ⚙️ **灵活配置**：支持自定义默认输入/输出路径，一次校准永久使用

## 🚀 快速开始

### 系统要求

- Windows 10/11
- Python 3.12+（开发环境）
- "码上放心"客户端（已安装并登录）

### 安装使用

#### 方式一：使用预编译版本（推荐）

1. 下载 `药品批发企业追朔码自动处理软件.exe`
2. 放置到 `E:\药品批发企业追朔码自动处理软件\` 目录
3. 创建桌面快捷方式
4. 双击运行

#### 方式二：从源码运行

```bash
# 克隆仓库
git clone https://github.com/lten44/drug-trace-automation.git
cd drug-trace-automation

# 安装依赖
pip install -r requirements.txt

# 运行
python gui.py
```

### 首次使用步骤

1. **校准位置**：点击"校准位置"按钮，5个校准点为：
   - 追溯码输入框
   - 查询按钮
   - 一键复制所有码（首次位置）
   - 一键复制所有码（查询后位置，v2.0新增）
   - 产品批号文字位置

2. **选择文件**：点击"浏览"选择要处理的Excel文件

3. **开始处理**：点击"开始处理"按钮，等待完成。码包装数为1的行会被自动跳过，大幅加速处理。

## 🏗️ 技术方案

### 方案选型

| 方案 | 可行性 | 结论 |
|------|--------|------|
| 码上放心开放API | ❌ 批发企业无接口权限 | 不可行 |
| WES付费接口 | ⚠️ 年费2800元 | 成本过高 |
| pywinauto控件定位 | ❌ WebView应用无法识别控件 | 不可行 |
| **pyautogui坐标模拟** | ✅ 屏幕坐标校准实现自动化 | ✅ **最终方案** |

### 核心设计：屏幕坐标校准系统

由于"码上放心"基于Chromium WebView，传统控件定位方案失效。通过**5点坐标校准**解决：

- 后台线程实时跟踪鼠标位置，F5热键记录坐标
- 校准数据持久化到 `calibration.json`，重启自动加载
- 双位置校准（v2.0）：复制按钮在查询前后位置不同，需分别记录

### 1级码过滤算法

剪贴板内容包含多级追溯码，通过标记检测精确提取1级码：

1. 找到"1级码"标记行
2. 持续收集后续所有纯数字行（16-24位）
3. 遇到其他级别标记时停止

### 智能跳过机制（v2.1）

如果输入表格中"码包装数"列值为1，说明该行本身就是1级码，无需查询关联关系。此时只获取产品批号，跳过完整的查询→复制→过滤流程。表格中1级码占比越高，提速越明显。
