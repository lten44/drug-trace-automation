## v3.0 大版本更新

### 核心逻辑重构
- **统一排重模式**：移除全1级/混装两模式，所有追溯码走同一流程
- **输入表匹配规则**：1级码查到同箱码后，只保留输入表中也存在的
- **零重复输出**：covered_set 覆盖排重 + output_1level_set 防重复，混装不同批号不误跳

### 新增功能
- 5秒倒计时操作前提示
- 查询失败弹窗（跳过/重试/停止三选一）
- 每步操作自动激活码上放心窗口（防焦点丢失）
- 空白码包装数自动标记异常
- 断点续跑（_progress.json 进度记录）

### Bug 修复
- F5校准：pynput + WinAPI 双保险全局监听 + 500ms 防抖
- 失败弹窗居中显示，不再卡死主窗口
- 校准窗口按一次F5不再连跳

### UI 美化
- Indigo 主色调 + 卡片左侧彩色装饰条（橙/蓝/绿区分功能）
- 按钮禁用时文字仍清晰可见（disabledforeground）
- 8px 圆角进度条
- Win7-Win11 全兼容
